# Customer Segmentation using Clustering

Project Overview:
This project performs customer segmentation using:
- K-Means Clustering
- Hierarchical Clustering

Dataset:

Marketing Campaign dataset

Steps:

1. Data Cleaning
2. Feature Engineering (Total_Spending, etc.)
3. Feature Scaling
4. Clustering (K-Means & Hierarchical)
5. Evaluation (Silhouette Score)

Results:

- K-Means best k = 2 → silhouette ≈ 0.40
- Hierarchical best k = 2 → silhouette ≈ 0.36
Customers are mainly divided into;
- Low-value customers
- High-value customers

Technologies:

- Python
- Pandas
- Scikit-learn
- Matplotlib
